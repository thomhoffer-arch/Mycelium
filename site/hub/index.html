hub

The connector registry.

The hub is the public index of conformant connectors. Each connector lives in its own repo on its own cadence; the hub just links them so people can find them. No central approval, no gatekeeper.

Suggest a source

Have a tool you'd like to see on the network — a CDE, a fabrication system, an in-house cost tracker — but you're not going to build the connector yourself? Open a suggestion. It tells the community what's wanted, and someone may pick it up.

Add your own connector

If you've built (or are building) a connector that passes the conformance kit, get it on the registry. The process is one PR:

  1. Make sure your connector declares the spine version it targets in its README.
  2. Run the conformance kit; ship a passing sample.
  3. Open a PR adding a row to hub/REGISTRY.md — name, source, spine version, repo URL, licence, maintainer, status.

The hub linter (run in CI) checks the row is well-formed. If something's off, the PR won't merge — but the error message tells you exactly what to fix.
