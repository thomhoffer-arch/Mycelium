The intelligence layer.

Mycelium is the open spine that any tool can speak. The intelligence layer is what runs on top — the program that joins the records, follows the identity edges, and proposes writes that a human approves before they're executed. The spine makes that layer possible (and interchangeable); the spine itself does not pretend to be one.

The first reference implementation is in development. The spec it consumes is open today, so anyone — open source, commercial, in-house — can build their own.
